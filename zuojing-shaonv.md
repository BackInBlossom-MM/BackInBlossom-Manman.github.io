# 从重度恋爱脑到“修无情道”：更年期老阿姨的作精少女黑历史

今天随意说些大学时的旧事作为催眠。

大学时老阿姨是个古怪性子，社团一律不参加，学校活动通通免谈，因着大一大二无所事事，着实好好学习了一阵子，核心专业课以及高数的成绩都实在不坏。

当然说到这里必需有个转折。

### 一、 千禧年初的“大户室”与我的金融启蒙

大二下学期我们学院有个大型活动搞证券节，当时股市还不普及，各大证券公司还有所谓的大户室。作为两财一贸的金融学院，不知是哪家金主赠送了学校一个电子大屏，而之前的信息学院师兄写了一个证券撮合交易的程序。

从此年年把机房的旧电脑搬出来，接上电子大屏，真金白银的收了各位同学的开户资金，假戏真做的模拟证券交易市场，还让各个系组织成基金机构作空或是做庄。拿到今天不算稀奇，可是千禧年初对于刚上大学的老阿姨们算是新鲜有趣。

不知为何从不参与学校任何活动的老阿姨被拉去输单子，大致是因为硬性摊派，每个班级的学习委员都被派了活，老阿姨当时十分不幸的被抓了壮丁。

前文说到老阿姨曾经看上一位大四快毕业的师兄就是这个活动，老阿姨被分到这位师兄分管的上市部打杂，觉得这位仁兄虽其貌不扬，但性格十分有趣。兼之当时大四的诸位师兄师姐均已找好去处，心中大石已落，于学校诸事颇为潇洒随意。

这位算是直属上司的师兄不耐对老阿姨科普股票事宜，遂荐了刘鸿儒的金融学和金融市场学，作为考五道口的必读教材，当然鞭辟入里，深入浅出。 这里的五道口不是今日的五道口男子技工学院，而是人民银行总行研究生部，彼时尚未并入清华，号称中国金融界黄浦军校。

当然说回这本书，刚学了货币银行学的老阿姨读金融界奠基大佬的书自然是惊为天人，故此对那位师兄印象大好。回去自然是如此这般的与我那位混迹学生会的美貌闺蜜一番私语。

### 二、 闺蜜暗渡陈仓与暴躁理工直男的登场

还没等我对那位有趣兼有才的大四师兄采取行动就得闻噩耗，这位师兄早已名草有主，虽不很甘愿，但老阿姨向来是有原则的人。

而没过几天闺蜜告知她已得手，老阿姨大为惊讶，师姐们的墙角都是这么好撬的吗？悔之已晚。

这里就轮到说队友了，队友当时在第一波千禧年计算机泡沫时混迹于各个风投小公司打工混经验，做各种粗制滥造的小网站。好巧不巧，当年证券节的撮合交易程序交予队友负责。队友整日在二楼调程序，我整日在二楼输单子兼听队友和他的上司理论程序跑不通的原因，心中暗暗作想脾气这么差以后指定找不到女朋友。

不料没过几天闺蜜就把队友推销给了我，老阿姨之所以半推半就的答应下来，一方面是闺蜜说辞动人，等队友大四出国即可无痛分手；另一方面是千禧年时会敲代码算是很 cool 的事情。

彼时那位师兄尚未离校，我们四人常出去轧马路。当然明面上是闺蜜和师兄陪我和队友，实际是我和队友给师兄和闺蜜打掩护，好在师兄正牌女友的眼皮字底下暗渡陈仓。 当时那位师兄是广州人，介绍了一款学校附近的馆子拿手的湄州点心，对于苏子瞻的家乡风味，老阿姨自然印象大好。

说到这位师兄的正牌女友，是位可爱的美貌师姐，且对待这位师兄十分温柔耐心，回回聚餐都看到这位师姐忙着给师兄挑香菜剥虾剔鱼刺，可见那句妻不如妾，偷不如偷不着的古话再正确没有了。

在于队友处对象之前，每个周末老阿姨自然是骑着我的破自行车绕过大半个北三环去当时的北图，如今的国图，消磨上两天。认识队友之后，日日队友不是编程就是啃砖头一样厚的计算机编程或是数据库的书，老阿姨百无聊赖的用队友的电脑看各种搞笑 flash。

### 三、 傲娇少女、恋爱脑与奇怪的人工智能博士

虽然如今老阿姨修无情道，年轻时是个不折不扣的恋爱脑，研究生时队友已经开始读博。

说到研究生不免说一下当年考研，考研一般大二就要开始准备，彼时尚没有网课，要挤很远的公交去上考研班。可整个大三的考研班统统听了个寂寞，人去了心不在。而最拿手的专业课，为了选导师时能跟上一众院长校长，选了最压分的方向，并且考试之前划重点的专业课辅导也没去听。

之所以后来能险险过线，全靠从小就苦练的必杀技短期记忆，大四临考前把各大考研班的政治材料苦背了一遍，政治考了 87 分。 结果选导师并没有按方向分，苦心孤诣完全白费。

说到导师，我确实算得上我当年研究生导师的开山大弟子。可实在十分愧对他老人家，不仅他研究生一入学就交给我的许多专业书并没有好好看，研究生毕业时还不假思索的推掉了他荐我去他在南大当博导的同门那里读博的一番殷切好意。毕业许多年并没有联系过导师，而读二硕时需要推荐信时贸然发了邮件居然得了回音，实属意外。

说到研究生时期，如今老阿姨是个修无情道的心如止水更年期大妈，当年却是个作天作地的傲娇文青少女加恋爱脑晚期重症。

傲娇少女是个什么性子？举个例子，我想吃糖，我不会管你要，你必须猜到我想吃糖，还要精准挑中我最喜欢的口味，剥好糖纸毕恭毕敬的放到我手心。

恋爱脑是个什么做派？每段时间不闹上十回八回分手复合的戏码就不叫谈恋爱。每回闹分手，就一边听陈奕迅的十年，一边给队友写小作文，当年写小作文掉书袋可比如今写这些更年期呓语厉害多了，从皎如天上月皑如山上雪的闻君有两意，到千金难买相如赋的长门怨统统来上一遍。 可怜队友是个只知道 if else 的编程理工直男，当初怎么熬过来的我都不能想像。

以致于作得太厉害我研究生时期从闺蜜到同学统统劝分不劝和。毕竟我当时是贸大女研，队友在英国不知名大学读一个不知道能不能毕业的奇怪人工智能博士。不过现如今队友的这个奇怪专业支撑了我们家大部分现金流，老阿姨的工钱只能略做贴补。

### 四、 古典BBS的灌水岁月与留不住的朱颜

老阿姨恋爱脑稍微好转，就迷上了在 bbs 灌水，研三的时候天天在小天鹅上和一众师弟师妹闲扯倚老卖老，还要求站长不得关我小黑屋。当时的站长 id 叫大鱼，出自大鱼海棠，具体是什么意思我现在都不知道。

有一个当时就十分具有时代眼光的师妹在考 acca，她的 id 叫 minicoper，所以如果老阿姨会开车一定对这款情有独钟；还有一个家庭条件颇好的师弟叫 bearichman，这个 id 被我们当时暗暗嘲笑了很久。

老阿姨当时除了灌水还干一些例如挂帖子给那位南京学新闻的网友在贸大征女友。可惜贸大美女见惯春风秋月并不如老阿姨一般喜好蛇精脸的男同胞，虽然老阿姨拿出十分功底帖子写的花团锦簇还被挂了十大，可一位应征的师妹都没有。

夜已深，明日尚要早起去例行牙医检查。最后说一句：

> 最是人间留不住，朱颜辞镜花辞树。
