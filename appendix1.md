The data analysis and processing for this study were conducted using custom scripts written in both Python and JavaScript. Python was primarily used for the final data modelling. At the same time, JavaScript was utilised on the Google Earth Engine (GEE) platform for the initial extraction and processing of geographic and meteorological information. In preparing this dissertation, the author acknowledges the use of several AI-powered tools, including OpenAI's ChatGPT models (primarily versions 4.1 and 4.5), which assisted in the development of these scripts.

Google's Gemini (2.5 Pro) was used to help plan chapter outlines, assist in summarising academic literature for preliminary understanding, optimise the clarity of figure and table captions, and refine the wording of this AI use statement. Grammarly Premium was utilised to proofread and enhance the clarity of the primary written text. 

All analysis decisions, model specifications, interpretations, and conclusions are solely my own. Outputs from AI tools were used only as drafting or coding assistance and were verified and edited by me; no generated text or code was used without review. The Python scripts for data modelling, along with the key datasets generated from the GEE process, are provided as supplementary files with this submission.
